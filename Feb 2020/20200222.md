## branches!
### git clone [url] [alias]
- DL from remote (origin/master to master)
### git branch [name]
- create a branch as [name] apart from master
### git merge [name]
- merge [name] branch to main branch
### git rebase master
- re-base the ref point as the latest version of master
### git branch -d [name]
- remove no longer needed branches
